BoxingFitness APP 用户协议
版本：V1.4
生效日期：2026-05-16

# 1. 概述
本协议适用于您使用绍兴微迈科技有限公司提供的 BoxingFitness APP 及相关服务。您下载、安装、登录或使用本 APP，即表示您同意本协议。

# 2. 服务内容
BoxingFitness 当前提供：
- 智能拳击手套蓝牙扫描、左右手套配对连接、连接状态和电量显示；
- 灵敏度设置、训练开始/结束时的设备控制和蓝牙击中事件识别；
- 色彩涂鸦、情绪拳王、自由拳击、极速燃脂、燃脂挑战、燃脂陪练等训练模块；
- 训练记录、锻炼成果、榜单、个人中心、语言设置和数据同步服务。

# 3. 使用条件
- 当前 APP 安装后可直接使用，不再进行序列号或激活码校验。
- 需要智能手套参与的训练功能，应先在设置界面连接同一编号的左、右手套。
- 您应确保使用环境安全、身体状况适合运动，并根据自身能力合理训练。

# 4. 训练数据与榜单规则
- 拳击次数、左右手击中、训练时长、卡路里、燃脂量、阶段结果和计划结果可能用于训练报告、锻炼成果、个人统计和榜单排名。
- 榜单可能展示每日最佳拳击数、累计拳击数、累计训练时长、累计卡路里、累计燃脂量和段位等指标。
- 卡路里和燃脂量为估算结果，仅供训练反馈，不构成医疗、营养、康复、竞技认证或减重承诺。

# 5. 用户行为规范
您不得伪造训练数据、操纵榜单、冒用他人账号、干扰蓝牙设备通信、逆向工程 APP、破坏服务稳定性，或以其他方式损害用户、设备、服务器或运营者权益。

# 6. 蓝牙设备使用
- APP 通过蓝牙连接智能拳击手套，并根据设备返回的击中事件进行计数和训练判定。
- 训练开始、结束或倒计时归零时，APP 可能向设备发送陀螺仪开关、灵敏度等控制指令。
- 因设备电量、距离、遮挡、系统蓝牙状态、环境干扰或硬件故障导致的连接异常，可能影响训练体验。

# 9. 免责声明
BoxingFitness 用于拳击训练、情绪放松和健身燃脂反馈，不替代医生、教练或专业机构意见。您应根据自身身体状况训练，并自行承担不当运动带来的风险。
